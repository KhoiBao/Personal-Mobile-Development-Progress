# Project Structure: BTVN1

BTVN1/                     ← **main project folder (root)**
│
├── bin/                   ← where main executable files go (for running apps)
│   └── btvn1.dart         
│
├── lib/                   ← where your library or reusable code lives
│   └── btvn1.dart         
│
├── test/                  ← for automated test files
│   └── btvn1_test.dart    
│
├── pubspec.yaml           ← like a “package.json” — describes your project & dependencies
│
├── pubspec.lock           ← locked versions of packages (auto-generated)
│
├── README.md              ← documentation / description    
│
├── CHANGELOG.md           ← version history / changes between version
│
├── analysis_options.yaml  ← linter / code style rules
│
└── .gitignore             ← tells Git which files to skip
